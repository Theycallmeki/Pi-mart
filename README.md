Pi-mart is our thesis project, a comprehensive general-purpose retail management system designed to help store owners make smarter, data-driven decisions through powerful analytics and business insights. The platform includes key features such as a point-of-sale (POS) system for seamless transactions, robust inventory management for real-time stock tracking, and AI-driven analytics that provide insights into sales trends, customer behavior, and overall store performance.

To deliver a complete ecosystem, Pi-mart provides two dedicated web applications: a Vue.js interface for store owners to efficiently manage operations, and a React.js application for customers to browse products and interact with the store. The system is built using Flask as the backend framework and is deployed using DigitalOcean and Docker, ensuring scalability, reliability, and ease of maintenance.
